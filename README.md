# IS-Project-Management

## Project Overview
This repository contains all project management artifacts for the **Online Course Registration System** – a web-based application that allows students to browse, add, and drop courses online.

## 📁 Repository Contents
| File | Description |
|------|-------------|
| `Project_Proposal.md` | Initial project proposal with objectives, scope, and timeline |
| `Project_Plan.md` | Detailed work breakdown structure and milestones |
| `Risk_Management.md` | Risk register with mitigation strategies |
| `Final_Report.md` | Post-project analysis and lessons learned |

## 🚀 Quick Start
1. Clone this repository
2. Open any `.md` file in a Markdown viewer or text editor
3. Review project documentation
